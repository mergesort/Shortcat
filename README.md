## Installation
You can use [CocoaPods](http://cocoapods.org/) to install `Shortcat` by adding it to your `Podfile`:

```swift
platform :ios, '9.0'
use_frameworks!

pod 'Shortcat'
```

Or install it manually by downloading the Swift files in the `Source` folder, and dropping them in your project.
